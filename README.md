# CONOCIENDO NUESTRAS CAJA_COOPERATIVA(SECTOR MICROFINANCIERO)

## ¿De qué trata el proyecto?

Elabora un sitio interactivo donde cada uno puedo conocer mas sobre CAJAS Y COOPÉRATIVAS

## ¿Por qué es importante? 

La educación financiera es un reto dentro de los OBjetivos de Desarrollo Sostenible, mediante esta iniciativa estamos tratando de cerrar esta brecha,donde cada uno pueda conocer los productos que brindan cada uno de los entidades financieras: cajas y cooperativas, como también las cifras mensuales(créditos, colocaciones, participación por tipo de crédito, numero de deudores, utlidades, ROE, morosidad, provisiones y tasa de interés )
